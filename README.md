# 2026广州中考报考指南查询系统

基于《2026广州市中考报考指南》PDF（174页）提取的完整学校数据查询系统。

## 📊 数据统计

| 指标 | 数值 |
|------|------|
| 学校总数 | 1,091 |
| 专业总数 | 1,831 |
| 有专业学校 | 473 |

## 🎯 功能

- 🔍 搜索学校名/专业名
- 🏛️ 公办/民办筛选
- 🗺️ 按区域筛选（天河、海珠、荔湾、越秀、白云、黄埔、番禺、花都、增城、从化、南沙）
- 📋 按批次筛选（第三批次、第四批次普通高中、中职三二分段、第四批次中职等）
- 📚 点击学校展开专业列表（代号、名称、学费、住宿费、招生计划、备注）
- 📱 手机响应式

## 🚀 使用

直接用浏览器打开 `index.html` 即可，零依赖。

## 📖 数据来源

- 《2026广州市中考报考指南》
- 部分缺地址学校标注为"详见报考指南原文"

## ⚙️ 技术

- 单HTML文件，所有CSS/JS内联
- 深色GitHub风格主题
- PyMuPDF提取PDF数据
